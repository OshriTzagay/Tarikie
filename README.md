# Tarikie - ታሪኬ | ספרי מורשת

## תיאור הפרויקט / የፕሮጀክት መግለጫ

Tarikie הוא אתר מודרני להצגת וקידום ספרי מורשת עבריים ואמהריים. הפרויקט מציע חוויית משתמש דו-לשונית מלאה עם תמיכה בעברית (RTL) ואמהרית.

Tarikie የዕብራይስጥ እና የአማርኛ ቅርስ መጽሐፍትን ለማሳየት እና ለማስተዋወቅ ዘመናዊ ድህረ ገጽ ነው። ይህ ፕሮጀክት ሙሉ ባለ ሁለት ቋንቋ የተጠቃሚ ተሞክሮ ከዕብራይስጥ (RTL) እና አማርኛ ድጋፍ ጋር ይሰጣል።

## טכנולוגיות / ቴክኖሎጂዎች

הפרויקט בנוי עם:
- **Vite** - כלי Build מהיר
- **TypeScript** - שפת תכנות מסוג
- **React 18** - ספריית UI
- **shadcn/ui** - קומפוננטות UI מודרניות
- **Tailwind CSS** - עיצוב מהיר ורספונסיבי
- **Framer Motion** - אנימציות מתקדמות
- **React Router** - ניווט בין עמודים
- **i18next** - תמיכה רב-לשונית

## התקנה והפעלה / ጭነት እና አሰራር

```sh
# התקנת תלויות / ጥገኝነቶችን ይጫኑ
npm install

# הפעלת שרת פיתוח / የልማት አገልጋይን ያስጀምሩ
npm run dev

# בנייה לפרודקשן / ለምርት ይገንቡ
npm run build

# הרצת טסטים / ፈተናዎችን ያስኬዱ
npm test
```
